# Mandarin — кулинарный сайт ресторана (HTML + CSS)

Тема проекта: кулинарный сайт ресторана «Mandarin» (Астана).
На сайте представлены разделы меню и страница рецептов, где каждая позиция из меню ведёт на подробную технологию приготовления.

## Страницы
- index.html — главная
- about.html — о ресторане
- menu.html — меню (все позиции кликабельны)
- recipes.html — рецепты (якоря по каждому блюду)
- contacts.html — контакты и поддержка

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox / Grid, адаптивность)

## Структура проекта
- css/reset.css — сброс стилей
- css/base.css — базовые стили, цвета, типографика
- css/layout.css — сетка и структура страниц
- css/components.css — кнопки, карточки и компоненты
